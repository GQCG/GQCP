\doxysubsection{Detailed Description}
A Cartesian Gaussian-\/type orbital (GTO), often referred to as a Cartesian \textquotesingle{}primitive\textquotesingle{}.

\begin{DoxyNote}{Note}
Mathematically speaking, a Cartesian GTO is a real-\/valued scalar function taking an Euclidean vector (3D-\/vector) as argument, which is why we inherit from \mbox{\hyperlink{class_g_q_c_p_1_1_scalar_function}{Scalar\+Function}}. 

Calling operator() returns the value of the unnormalized Cartesian Gaussian. 

Contracted GTOs can be expressed as linear combinations of GTOs\+: Linear\+Combination$<$\+Cartesian\+GTO$>$. 
\end{DoxyNote}
