A restricted second-\/quantized one-\/electron operator whose parameters are evaluatable at a point in space.


\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+\+Function\+Type} & The type of evaluatable function that is used as a \textquotesingle{}matrix element\textquotesingle{} of this one-\/electron operator. \\
\hline
{\em \+\_\+\+Vectorizer} & The type of the vectorizer that relates a one-\/dimensional storage of matrices to the tensor structure of one-\/electron operators. This allows for a distinction between scalar operators (such as the density) and vector operators (such as the current density operator). \\
