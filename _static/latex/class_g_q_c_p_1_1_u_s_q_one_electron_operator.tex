\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Scalar, typename \+\_\+\+Vectorizer$>$\newline
class G\+Q\+C\+P\+::\+U\+S\+Q\+One\+Electron\+Operator$<$ \+\_\+\+Scalar, \+\_\+\+Vectorizer $>$}

A class that represents an \textquotesingle{}unrestricted second-\/quantized one-\/electron operator\textquotesingle{}. This type of operator is suitable for the projection of the non-\/relativistic Hamiltonian onto an unrestricted spinor basis. It holds the matrix representation of its parameters for both spin components.


\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+\+Scalar} & The scalar type used for a single parameter\+: real or complex. \\
\hline
{\em \+\_\+\+Vectorizer} & The type of the vectorizer that relates a one-\/dimensional storage of matrices to the tensor structure of one-\/electron operators. This allows for a distinction between scalar operators (such as the kinetic energy operator), vector operators (such as the spin operator) and matrix/tensor operators (such as quadrupole and multipole operators). \\
\hline
\end{DoxyTemplParams}
