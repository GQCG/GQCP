\doxysubsection{Detailed Description}
A London-\/modified Cartesian GTO.

\begin{DoxyNote}{Note}
Mathematically speaking, a Cartesian GTO is a complex-\/valued scalar function taking an Euclidean vector (3D-\/vector) as argument, which is why we inherit from {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_function}{Function}}}. 

Calling operator() returns the value of the unnormalized London GTO. 

Contracted London GTOs can be expressed as linear combinations of GTOs\+: {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_evaluable_linear_combination}{Evaluable\+Linear\+Combination}}\texorpdfstring{$<$}{<}\mbox{\hyperlink{class_g_q_c_p_1_1_london_cartesian_g_t_o}{London\+Cartesian\+GTO}}\texorpdfstring{$>$}{>}}. 
\end{DoxyNote}
