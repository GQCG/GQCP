class G\+Q\+C\+P\+::\+R\+Transformation$<$ \+\_\+\+Scalar $>$}

A \textquotesingle{}restricted\textquotesingle{} basis transformation, i.\+e. a spin-\/orbital basis transformation where the transformation is applied equally to the alpha-\/ and beta-\/spin-\/orbitals.
