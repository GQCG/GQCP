\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Scalar, typename \+\_\+\+Vectorizer, typename \+\_\+\+Derived\+Operator$>$\newline
class GQCP\+::\+Simple\+SQTwo\+Electron\+Operator$<$ \+\_\+\+Scalar, \+\_\+\+Vectorizer, \+\_\+\+Derived\+Operator $>$}
A second-\/quantized one-\/electron operator whose parameters are described by a single tensor.

This class is used as a base class for {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_r_s_q_two_electron_operator}{RSQTwo\+Electron\+Operator}}} and {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_g_s_q_two_electron_operator}{GSQTwo\+Electron\+Operator}}}, since they both admit parameter representations using a single tensor, as opposed to {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_u_s_q_two_electron_operator}{USQTwo\+Electron\+Operator}}}, which uses separate alpha-\/ and beta-\/ tensors. The word \textquotesingle{}simple\textquotesingle{} is used here as an antonym for \textquotesingle{}compound\textquotesingle{}.
