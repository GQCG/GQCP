\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Scalar, typename \+\_\+\+Vectorizer, typename \+\_\+\+Derived\+Operator$>$\newline
class G\+Q\+C\+P\+::\+Simple\+S\+Q\+Two\+Electron\+Operator$<$ \+\_\+\+Scalar, \+\_\+\+Vectorizer, \+\_\+\+Derived\+Operator $>$}

A second-\/quantized one-\/electron operator whose parameters are described by a single tensor.

This class is used as a base class for {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_r_s_q_two_electron_operator}{R\+S\+Q\+Two\+Electron\+Operator}}} and {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_g_s_q_two_electron_operator}{G\+S\+Q\+Two\+Electron\+Operator}}}, since they both admit parameter representations using a single tensor, as opposed to {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_u_s_q_two_electron_operator}{U\+S\+Q\+Two\+Electron\+Operator}}}, which uses separate alpha-\/ and beta-\/ tensors. The word \textquotesingle{}simple\textquotesingle{} is used here as an antonym for \textquotesingle{}compound\textquotesingle{}.


\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+\+Scalar} & The scalar type used for a single parameter/matrix element\+: real or complex. \\
\hline
{\em \+\_\+\+Vectorizer} & The type of the vectorizer that relates a one-\/dimensional storage of tensors to the tensor structure of two-\/electron operators. This distinction is carried over from \mbox{\hyperlink{class_g_q_c_p_1_1_simple_s_q_one_electron_operator}{Simple\+S\+Q\+One\+Electron\+Operator}}. \\
\hline
{\em \+\_\+\+Derived\+Operator} & The type of the operator that derives from this class, enabling \mbox{\hyperlink{class_g_q_c_p_1_1_c_r_t_p}{C\+R\+TP}} and compile-\/time polymorphism. \\
