\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Scalar, typename \+\_\+\+Derived\+Transformation$>$\newline
class G\+Q\+C\+P\+::\+Simple\+Transformation$<$ \+\_\+\+Scalar, \+\_\+\+Derived\+Transformation $>$}

A basis transformation that can be represented by a single transformation matrix.

In general, we adopt the convention outlined in (\href{https://gqcg-res.github.io/knowdes/spinor-transformations.html}{\texttt{ https\+://gqcg-\/res.\+github.\+io/knowdes/spinor-\/transformations.\+html}}), where the new orbitals\textquotesingle{} coefficients can be found in the respective {\bfseries{column}} of the related transformation matrix.

This class is used as a base class for {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_r_transformation}{R\+Transformation}}} and {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_g_transformation}{G\+Transformation}}}, since they are both expressed using a single matrix, as opposed to {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_u_transformation}{U\+Transformation}}}, which uses separate transformation coefficients for alpha-\/ and beta-\/ matrices. The word \textquotesingle{}simple\textquotesingle{} is used here as an antonym for \textquotesingle{}compound\textquotesingle{}.
