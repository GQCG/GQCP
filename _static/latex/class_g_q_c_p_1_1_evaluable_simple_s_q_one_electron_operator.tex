\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Function\+Type, typename \+\_\+\+Vectorizer, typename \+\_\+\+Derived\+Evaluable\+Operator$>$\newline
class GQCP\+::\+Evaluable\+Simple\+SQOne\+Electron\+Operator$<$ \+\_\+\+Function\+Type, \+\_\+\+Vectorizer, \+\_\+\+Derived\+Evaluable\+Operator $>$}

An evaluable, simple second-\/quantized one-\/electron operator whose parameters are evaluatable at a point in space.

This type (and its derived types) distinguish(es) itself from {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_simple_s_q_one_electron_operator}{Simple\+SQOne\+Electron\+Operator}}}, which is mainly to be used for scalar types that are real or complex, thus enabling conformance to {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_basis_transformable}{Basis\+Transformable}}} and {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_jacobi_rotatable}{Jacobi\+Rotatable}}}. On the contrary, this type does not provide these conformances and transformation formulas should be implemented ad-\/hoc.

This allows for a distinction between scalar operators (such as the density) and vector operators (such as the current density operator). \\
