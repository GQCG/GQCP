class G\+Q\+C\+P\+::\+Simple1\+D\+M$<$ \+\_\+\+Scalar, \+\_\+\+Derived\+D\+M $>$}

A one-\/electron density matrix that is described by a single matrix.

This class is used as a base class for {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_orbital1_d_m}{Orbital1\+DM}}} and {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_g1_d_m}{G1\+DM}}}, since they are both expressed using a single matrix, as opposed to {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_spin_resolved1_d_m}{Spin\+Resolved1\+DM}}}, which uses separate alpha-\/ and beta-\/ matrices. The word \textquotesingle{}simple\textquotesingle{} is used here as an antonym for \textquotesingle{}compound\textquotesingle{}.
