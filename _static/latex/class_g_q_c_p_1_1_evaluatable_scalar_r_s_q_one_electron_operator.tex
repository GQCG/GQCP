class G\+Q\+C\+P\+::\+Evaluatable\+Scalar\+R\+S\+Q\+One\+Electron\+Operator$<$ \+\_\+\+Evaluatable $>$}

A scalar restricted second-\/quantized one-\/electron operator whose parameters are evaluatable at a point in space.

This type distinguishes itself from {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_r_s_q_one_electron_operator}{R\+S\+Q\+One\+Electron\+Operator}}}, which is mainly to be used for scalar types that are real or complex, thus enabling conformance to {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_basis_transformable}{Basis\+Transformable}}} and {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_jacobi_rotatable}{Jacobi\+Rotatable}}}. On the contrary, this type does not provide these conformances and transformation formulas are implemented ad-\/hoc.
