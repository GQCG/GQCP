class G\+Q\+C\+P\+::\+Spin\+Resolved1\+D\+M\+Component$<$ \+\_\+\+Scalar $>$}

One of the spin components of a {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_spin_resolved1_d_m}{Spin\+Resolved1\+DM}}}.

It is specifically designed as one of these spin components, in order to ensuring compile-\/time correctness. It would be wrong to use either {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_orbital1_d_m}{Orbital1\+DM}}} or {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_g1_d_m}{G1\+DM}}} as one of the spin components, and it\textquotesingle{}s not possible to use {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_simple1_d_m}{Simple1\+DM}}} as one of the spin components because it requires a template argument of the type that derives from it.
