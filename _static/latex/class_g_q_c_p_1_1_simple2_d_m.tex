\subsubsection*{template$<$typename \+\_\+\+Scalar, typename \+\_\+\+Derived\+DM$>$\newline
class GQCP\+::\+Simple2\+DM$<$ \+\_\+\+Scalar, \+\_\+\+Derived\+DM $>$}
A two-\/electron density matrix that is described by a single tensor.

This class is used as a base class for {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_orbital2_d_m}{Orbital2\+DM}}} and {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_g2_d_m}{G2\+DM}}}, since they are both expressed using a single tensor, as opposed to {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_spin_resolved2_d_m}{Spin\+Resolved2\+DM}}}, which uses separate alpha-\/ and beta-\/ tensor. The word \textquotesingle{}simple\textquotesingle{} is used here as an antonym for \textquotesingle{}compound\textquotesingle{}.
