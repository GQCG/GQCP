class G\+Q\+C\+P\+::\+U\+Transformation\+Component$<$ \+\_\+\+Scalar $>$}

One of the spin components of a {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_u_transformation}{U\+Transformation}}}.

It is specifically designed as one of these spin components, in order to ensuring compile-\/time correctness. It would be wrong to use either R/\+G\+Transformation as one of the spin components, and it\textquotesingle{}s not possible to use \mbox{\hyperlink{class_g_q_c_p_1_1_simple_transformation}{Simple\+Transformation}} as one of the spin components because it requires a template argument of the type that derives from it.
