MARK\+: Simple\+Orbital\+Rotation\+Generator implementation A set of orbital rotation generators that can be represented by a single vector.

This class is used as a base class for {\ttfamily ROrbital\+Rotation\+Generator} and {\ttfamily GOrbital\+Rotation\+Generator}, since they are both expressed using a single vector of kappa\+\_\+\+PQ values, as opposed to {\ttfamily UOrbital\+Rotation\+Generator}, which uses separate vectors for alpha-\/ and beta-\/ generators. The word \textquotesingle{}simple\textquotesingle{} is used here as an antonym for \textquotesingle{}compound\textquotesingle{}.
