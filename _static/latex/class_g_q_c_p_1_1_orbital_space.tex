\doxysubsection{Detailed Description}
A class that encapsulates occupied, active and virtual orbital indices.

\begin{DoxyNote}{Note}
The union of these three sets of indices is supposed to be the full set of orbital indices. 

This class is intended to be used in conjunction with restricted spin-\/orbital bases (\mbox{\hyperlink{class_g_q_c_p_1_1_r_spin_orbital_basis}{R\+Spin\+Orbital\+Basis}}) or general spinor bases (\mbox{\hyperlink{class_g_q_c_p_1_1_g_spinor_basis}{G\+Spinor\+Basis}}). 
\end{DoxyNote}
