\begin{DoxyNote}{Note}
An integration grid (.igrid) file is a headerless file and contains the following data\+:
\begin{DoxyItemize}
\item Each row relates to one grid point.
\item Column specification\+:
\begin{DoxyItemize}
\item Column 1\+: The index from 1 to the number of grid points
\item Columns 2-\/4\+: The position of the grid point\+: x, y, and z
\item Optional\+: Column 5 or columns 5-\/7\+: 1 value for a scalar field, 3 values for a vector field
\item Column 5, 6 or 8\+: The integration weight associated to the grid point 
\end{DoxyItemize}
\end{DoxyItemize}
\end{DoxyNote}
