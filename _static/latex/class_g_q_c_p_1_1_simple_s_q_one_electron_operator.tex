\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Scalar, typename \+\_\+\+Vectorizer, typename \+\_\+\+Derived\+Operator$>$\newline
class GQCP\+::\+Simple\+SQOne\+Electron\+Operator$<$ \+\_\+\+Scalar, \+\_\+\+Vectorizer, \+\_\+\+Derived\+Operator $>$}
A second-\/quantized one-\/electron operator whose parameters are described by a single matrix.

This class is used as a base class for {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_r_s_q_one_electron_operator}{RSQOne\+Electron\+Operator}}}, {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_g_s_q_one_electron_operator}{GSQOne\+Electron\+Operator}}}, and {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_u_s_q_one_electron_operator_component}{USQOne\+Electron\+Operator\+Component}}}, since they both admit parameter representations using a single matrix, as opposed to the full {\ttfamily \mbox{\hyperlink{class_g_q_c_p_1_1_u_s_q_one_electron_operator}{USQOne\+Electron\+Operator}}}, which uses separate alpha-\/ and beta-\/ matrices. The word \textquotesingle{}simple\textquotesingle{} is used here as an antonym for \textquotesingle{}compound\textquotesingle{}.


\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+\+Scalar} & The scalar type used for a single parameter/matrix element\+: real or complex. \\
\hline
{\em \+\_\+\+Vectorizer} & The type of the vectorizer that relates a one-\/dimensional storage of matrices to the tensor structure of one-\/electron operators. This allows for a distinction between scalar operators (such as the kinetic energy operator), vector operators (such as the spin operator) and matrix/tensor operators (such as quadrupole and multipole operators). \\
\hline
{\em \+\_\+\+Derived\+Operator} & The type of the operator that derives from this class, enabling \mbox{\hyperlink{class_g_q_c_p_1_1_c_r_t_p}{CRTP}} and compile-\/time polymorphism. \\
