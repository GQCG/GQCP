Detailed Description

template<typename _Scalar, typename _Primitive>
class GQCP::CurrentDensityMatrixElement< _Scalar, _Primitive >

One of the elements of (one of the vector components of) the second-quantized field-free charge current density operator.

Parameters
_ScalarThe scalar type of the expansion coefficients of the spatial orbitals that underlie this current density matrix element.
_PrimitiveThe type of primitive that underlies this current density matrix element.
