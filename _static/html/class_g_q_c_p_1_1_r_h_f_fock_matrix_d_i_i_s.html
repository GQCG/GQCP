Detailed Description

template<typename _Scalar>
class GQCP::RHFFockMatrixDIIS< _Scalar >

An iteration step that accelerates the Fock matrix (expressed in the scalar/AO basis) based on a DIIS accelerator.

Template Parameters
_ScalarThe scalar type used to represent the expansion coefficient/elements of the transformation matrix: real or complex.
