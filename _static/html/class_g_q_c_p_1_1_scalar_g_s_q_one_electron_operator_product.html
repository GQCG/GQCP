Detailed Description

template<typename _Scalar>
class GQCP::ScalarGSQOneElectronOperatorProduct< _Scalar >

A type that encapsulates the matrix elements of the product of two scalar, generalized one-electron operators.

Template Parameters
_ScalarThe scalar type of one of the matrix elements: real or complex.
