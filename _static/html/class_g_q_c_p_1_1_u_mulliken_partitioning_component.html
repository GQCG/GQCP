template<typename _Scalar>
class GQCP::UMullikenPartitioningComponent< _Scalar >

One of the components of an unrestricted Mulliken-based partitioning of an AO basis.

Parameters
_ScalarThe scalar type used to represent an element of the Mulliken projection matrix: real or complex.
