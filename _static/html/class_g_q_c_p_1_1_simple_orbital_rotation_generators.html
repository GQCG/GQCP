Detailed Description

template<typename _Scalar, typename _DerivedOrbitalRotationGenerators>
class GQCP::SimpleOrbitalRotationGenerators< _Scalar, _DerivedOrbitalRotationGenerators >

MARK: SimpleOrbitalRotationGenerator implementation A set of orbital rotation generators that can be represented by a single vector.

This class is used as a base class for ROrbitalRotationGenerator and GOrbitalRotationGenerator, since they are both expressed using a single vector of kappa_PQ values, as opposed to UOrbitalRotationGenerator, which uses separate vectors for alpha- and beta- generators. The word 'simple' is used here as an antonym for 'compound'.

Template Parameters
_ScalarThe scalar type used for an orbital rotation generator: real or complex.
_DerivedOrbitalRotationGeneratorThe type of the orbital rotation generator that derives from this class, enabling CRTP and compile-time polymorphism.
