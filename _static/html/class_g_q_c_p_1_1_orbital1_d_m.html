Detailed Description

template<typename _Scalar>
class GQCP::Orbital1DM< _Scalar >

A type used to represent a one-electron orbital density matrix, i.e. the summed alpha and beta density matrix.

Template Parameters
_ScalarThe scalar type used for a density matrix element: real or complex.
Note
This type represents the (1/sqrt(2)-scaled) singlet (0,0) 1-DM.
