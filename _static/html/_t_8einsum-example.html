T.einsum

Contract this tensor with a matrix, using a NumPy 'einsum'-like API.

Parameters
contraction_stringThe string used to specify the wanted contraction, e.g. "ijkl,jk->il"
rhsThe right-hand side of the contraction, a matrix in this case.
Template Parameters
NThe number of axes that should be contracted over.

("ijkl,jk->il", M) will contract the j and k axes of the second tensor with those of the first tensor, resulting in a rank 2 tensor with axes i and l.

Returns
The result of the tensor contraction.
