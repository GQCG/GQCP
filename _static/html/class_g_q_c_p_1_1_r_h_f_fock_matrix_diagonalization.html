Detailed Description

template<typename _Scalar>
class GQCP::RHFFockMatrixDiagonalization< _Scalar >

An iteration step that solves the generalized eigenvalue problem for the current scalar/AO basis Fock matrix for the coefficient matrix.

Template Parameters
_ScalarThe scalar type used to represent the expansion coefficient/elements of the transformation matrix: real or complex.
