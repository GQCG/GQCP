Detailed Description

A Cartesian Gaussian-type orbital (GTO), often referred to as a Cartesian 'primitive'.

Note
Mathematically speaking, a Cartesian GTO is a real-valued scalar function taking an Euclidean vector (3D-vector) as argument, which is why we inherit from ScalarFunction.
Calling operator() returns the value of the unnormalized Cartesian Gaussian.
Contracted GTOs can be expressed as linear combinations of GTOs: LinearCombination<CartesianGTO>.
