Detailed Description

template<typename Scalar, typename Vectorizer>
struct GQCP::OperatorTraits< GSQOneElectronOperator< Scalar, Vectorizer > >

A type that provides compile-time information (traits) on GSQOneElectronOperator that is otherwise not accessible through a public class alias.

Template Parameters
ScalarThe scalar type used for a single parameter: real or complex.
VectorizerThe type of the vectorizer that relates a one-dimensional storage of matrices to the tensor structure of one-electron operators. This allows for a distinction between scalar operators (such as the kinetic energy operator), vector operators (such as the spin operator) and matrix/tensor operators (such as quadrupole and multipole operators).
