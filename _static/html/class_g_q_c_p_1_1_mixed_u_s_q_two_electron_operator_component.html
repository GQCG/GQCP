Detailed Description

template<typename _Scalar, typename _Vectorizer>
class GQCP::MixedUSQTwoElectronOperatorComponent< _Scalar, _Vectorizer >

One of the mixed (i.e. alpha-beta or beta-alpha) spin components of an unrestricted two-electron operator.

Template Parameters
_ScalarThe scalar type used for a single parameter: real or complex.
_VectorizerThe type of the vectorizer that relates a one-dimensional storage of matrices to the tensor structure of two-electron operators. This distinction is carried over from USQOneElectronOperator.
