Detailed Description

template<typename _Scalar>
class GQCP::MatrixRepresentationEvaluationContainer< Eigen::SparseMatrix< _Scalar > >

Sparse template specialization is required because insertions into an existing sparse matrix are expensive Elements should only be added to the matrix once all of them are evaluated in a vector of triplets

Template Parameters
_ScalarThe scalar type of one of the matrix elements: real or complex.
