Detailed Description

template<typename _ExpansionScalar, typename _FinalSpinorBasis>
class GQCP::SimpleSpinorBasis< _ExpansionScalar, _FinalSpinorBasis >

A class that represents a spinor basis that has no internal structure (hence 'simple') with respect to spin components.

Template Parameters
_ExpansionScalarThe scalar type used to represent an expansion coefficient of the spinors in the underlying scalar orbitals: real or complex.
_FinalSpinorBasisThe spinor basis that ultimately derives from this class, enabling CRTP and compile-time polymorphism.
