Detailed Description

template<typename _Scalar>
class GQCP::MixedSpinResolved2DMComponent< _Scalar >

One of the mixed (i.e. alpha-beta or beta-alpha) spin components of a spin-resolved 2-DM.

Template Parameters
_ScalarThe scalar type used for a density matrix element: real or complex.
