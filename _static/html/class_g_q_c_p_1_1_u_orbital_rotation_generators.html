Detailed Description

template<typename _Scalar>
class GQCP::UOrbitalRotationGenerators< _Scalar >

Template Parameters
_ScalarThe scalar type used for an orbital rotation generator: real or complex.

inlinestatic

Construct orbital rotation generators by adding redundant (i.e. 0) generators to the given occupation_type - occupation_type generators. This is done for the alpha and beta component separately.

Parameters
generatorsThe orbital rotation generators of the specified ocupation types.
row_occupation_typeThe occupation type of the rows of the orbital rotation generator kappa matrix.
column_occupation_typeThe occupation type of the column of the orbital rotation generator kappa matrix.
KThe total number of orbitals. In unrestricted and unrestricted these will be spin-orbitals.
Returns
The 'full' orbital rotation generators from the given row_occupation_type - column_occupation_type generators, separately constructed for the alpha and beta component.
