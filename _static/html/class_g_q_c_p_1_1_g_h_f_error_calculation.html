Detailed Description

template<typename _Scalar>
class GQCP::GHFErrorCalculation< _Scalar >

An iteration step that calculates the error matrix from the Fock and density matrices (expressed in the scalar/AO basis).

Template Parameters
_ScalarThe scalar type used to represent the expansion coefficient/elements of the transformation matrix: real or complex.
