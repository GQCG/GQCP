Construct a mixed-spin component of the spin-conserved stability matrix A'.

Note
The formula for this component is as follows: A'_I(sigma)A(sigma)J(sigma_bar)B(sigma_bar) = (A(sigma)I(sigma)|J(sigma_bar)B(sigma_bar)).
Sigma_bar = alpha if sigma is beta and vice versa.
The name Spin-conservedcomes from the article "Constraints and stability in Hartree-Fock theory" by Seeger, R. and Pople J.A. (https://doi.org/10.1063/1.434318).
Parameters
usq_hamiltonianThe second quantized Hamiltonian, expressed in the orthonormal, 'unrestricted' spin orbital basis of the UHF MOs, which contains the necessary two-electron operators.
sigmaThe spin-sigma component.
sigma_barThe spin component, opposite to sigma.
Returns
The mixed spin sigma component of stability matrix A'.

Construct a pure-spin component of the spin-conserved stability matrix A'.

Note
The formula for this component is as follows: A'_I(sigma)A(sigma)J(sigma)B(sigma) = \delta_I(sigma)J(sigma) * F_B(sigma)A(sigma) - \delta_B(sigma)A(sigma) * F_I(sigma)J(sigma) + (A(sigma)I(sigma)||J(sigma)B(sigma)).
The name Spin-conservedcomes from the article "Constraints and stability in Hartree-Fock theory" by Seeger, R. and Pople J.A. (https://doi.org/10.1063/1.434318).
Parameters
usq_hamiltonianThe second quantized Hamiltonian, expressed in the orthonormal, 'unrestricted' spin orbital basis of the UHF MOs, which contains the necessary two-electron operators.
sigmaThe spin-sigma component. This method creates a pure spin component, so all sigma's in the formula are either alpha or beta.
Returns
The pure spin sigma component of stability matrix A'.

Construct a component of the spin-unconserved stability matrix A''.

Note
The formula for this component is as follows: A''_I(sigma)A(sigma_bar)J(sigma)B(sigma_bar) = \delta_I(sigma)J(sigma) * F_B(sigma_bar)A(sigma_bar) - \delta_B(sigma_bar)A(sigma_bar) * F_I(sigma)J(sigma) - (A(sigma_bar)B(sigma_bar)|J(sigma)I(sigma)).
Sigma_bar = alpha if sigma is beta and vice versa.
The name Spin-unconservedcomes from the article "Constraints and stability in Hartree-Fock theory" by Seeger, R. and Pople J.A. (https://doi.org/10.1063/1.434318).
Parameters
usq_hamiltonianThe second quantized Hamiltonian, expressed in the orthonormal, 'unrestricted' spin orbital basis of the UHF MOs, which contains the necessary two-electron operators.
sigmaThe spin-sigma component.
sigma_barThe spin component, opposite to sigma.
Returns
A component of stability matrix A''.

The mixed alpha-beta two electron integrals are extracted from the Hamiltonian.
