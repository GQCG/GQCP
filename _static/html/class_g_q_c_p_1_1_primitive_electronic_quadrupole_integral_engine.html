Detailed Description

template<typename _Shell>
class GQCP::PrimitiveElectronicQuadrupoleIntegralEngine< _Shell >

A class that can calculate electronic dipole integrals, i.e. over the negative of the position operator.

Template Parameters
_ShellThe type of shell that this integral engine is related to.
Note
The integrals that this primitive engine produces include the minus sign due to the charge of the electron.
