Detailed Description

template<size_t N, typename _Scalar, typename _Vectorizer>
class GQCP::BaseFQOperator< N, _Scalar, _Vectorizer >

A base class used to represent first-quantized one-electron operators.

Template Parameters
_NThe number of electrons related to this operator: usually 1 or 2.
_ScalarThe scalar representation of the operator: real or complex.
_VectorizerThe type of the vectorizer that relates to the tensor structure of one-electron operators. This allows for a distinction between scalar operators (such as the kinetic energy operator), vector operators (such as the spin operator) and matrix/tensor operators (such as quadrupole and multipole operators).
