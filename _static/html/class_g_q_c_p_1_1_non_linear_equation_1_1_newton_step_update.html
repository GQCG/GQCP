Detailed Description

template<typename _Scalar, typename _Environment, typename LinearSolver = decltype(LinearEquationSolver<_Scalar>::ColPivHouseholderQR())>
class GQCP::NonLinearEquation::NewtonStepUpdate< _Scalar, _Environment, LinearSolver >

An iteration step that produces updated variables according to a Newton step.

Template Parameters
_Scalarthe scalar type that is used to represent the variables of the system of equations
_Environmentthe type of the calculation environment
_LinearSolverthe type of the solver that performs the Newton step, this defaults to the type of the ColPivHouseholderQR linear equations solver
