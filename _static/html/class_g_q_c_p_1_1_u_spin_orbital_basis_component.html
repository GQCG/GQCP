Detailed Description

template<typename _ExpansionScalar, typename _Shell>
class GQCP::USpinOrbitalBasisComponent< _ExpansionScalar, _Shell >

A type specifically designed as one of the spin-components of a USpinOrbitalBasis.

Template Parameters
_ExpansionScalarThe scalar type used to represent an expansion coefficient of the spin-orbitals in the underlying scalar orbitals: real or complex.
_ShellThe type of shell the underlying scalar basis contains.
