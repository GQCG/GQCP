Detailed Description

template<size_t N, typename _Scalar>
class GQCP::BaseScalarFQOperator< N, _Scalar >

A base class for scalar operators. Since the underlying Vectorizer is ScalarVectorizer, we're sure -at compile time- that the number of components is 1.

Template Parameters
_NThe number of electrons related to this operator: usually 1 or 2.
_ScalarThe scalar representation of the operator: real or complex.
