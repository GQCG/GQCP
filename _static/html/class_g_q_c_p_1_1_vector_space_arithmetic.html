Detailed Description

template<typename T, typename Scalar>
class GQCP::VectorSpaceArithmetic< T, Scalar >

An (abstract) interface for types that support vector space arithmetic, i.e. linear combinations.

Parameters
TThe type for which vector space arithmetic should be implemented. It is given as a template argument, enabling CRTP.
ScalarThe scalar type (real or complex) of the scalar multiplication.
