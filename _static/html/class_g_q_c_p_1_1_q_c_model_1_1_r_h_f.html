inline

Construct the singlet A stability matrix from the RHF stability conditions.

Note
The formula for the singlet A matrix is as follows: A_IAJB = \delta_IJ * (F_R)_BA - \delta_AB * (F_R)_IJ + 2 * (AI|JB) - (AB|JI).
Parameters
rsq_hamiltonianThe second quantized Hamiltonian, expressed in the orthonormal, 'restricted' spin orbital basis of the RHF MOs, which contains the necessary two-electron operators.
Returns
The singlet-A stability matrix.
