T1.einsum

Contract this tensor with another one, using a NumPy 'einsum'-like API.

Parameters
rhsThe right-hand side of the contraction.
lhs_labelsThe labels for the axes of the tensor on the left-hand side of the contraction.
rhs_labelsThe labels for the axes of the tensor on the right-hand side of the contraction.
output_labelsThe labels for the the resulting/output tensor.
Template Parameters
NThe number of axes that should be contracted over.

(T2, 'ijkl', 'ia', 'jkla') will contract the first axis of T1 (with labels 'ijkl') with the first axis of T2 (with labels 'ia') (because of the matching index labels 'i') and return a tensor whose axes are labelled as 'jkla'.

Returns
The result of the tensor contraction.
