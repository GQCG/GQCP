Detailed Description

The first-quantized, molecular electronic Hamiltonian for systems in a magnetic field.

Note
The molecular magnetic Hamiltonian is different from the molecular Pauli Hamiltonian: the Pauli Hamiltonian contains the additional paramagnetic spin Zeeman term.
