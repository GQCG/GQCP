Calculate the overlap <on|of>: the projection of between this spin-resolved ONV ('of') and another spin-resolved ONV ('on'), expressed in different R/U-spinor bases. The 'on'-ONV is supposed to be expressed in restricted spin-orbitals, and the 'of'-ONV is supposed to be expressed in unrestricted spin-orbitals.

Parameters
onv_onThe spin-resolved ONV that should be projected on.
C_unrestrictedThe transformation between the unrestricted spin-orbitals and the atomic spin-orbitals.
C_restrictedThe transformation between the restricted spin-orbitals and the atomic spin-orbitals.
SThe overlap matrix of the underlying AOs.
Returns
The overlap element <on|of>.

method can be used to project UHF-ONVs onto RHF-ONVs, by calling uhf_onv.calculateProjection(rhf_onv, C_unrestricted, C_restricted, S)
