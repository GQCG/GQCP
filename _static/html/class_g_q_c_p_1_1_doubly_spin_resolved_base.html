Detailed Description

template<typename _Pure, typename _Mixed, typename _Derived>
class GQCP::DoublySpinResolvedBase< _Pure, _Mixed, _Derived >

A utility type encapsulating four objects, each for every combination of alpha and beta.

Parameters
_PureThe type that represents a 'pure' combination of spin components, i.e. the alpha-alpha or beta-beta type.
_MixedThe type that represents a 'mixed' combination of spin components, i.e. the alpha-beta or beta-alpha type.
_DerivedThe type that derives from this type, given as a template argument, enabling CRTP and compile-time polymorphism.
