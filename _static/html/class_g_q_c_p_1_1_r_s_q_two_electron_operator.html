Detailed Description

template<typename _Scalar, typename _Vectorizer>
class GQCP::RSQTwoElectronOperator< _Scalar, _Vectorizer >

A restricted two-electron operator, which is suited for expressing non-relativistic (spin-free) two-electron operators.

Template Parameters
_ScalarThe scalar type used for a single parameter/matrix element: real or complex.
_VectorizerThe type of the vectorizer that relates a one-dimensional storage of tensors to the tensor structure of two-electron operators. This distinction is carried over from SimpleSQOneElectronOperator.
