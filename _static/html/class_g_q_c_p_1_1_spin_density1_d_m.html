Detailed Description

template<typename _Scalar>
class GQCP::SpinDensity1DM< _Scalar >

A type used to represent a one-electron spin-density density matrix, i.e. the alpha density matrix minus the beta density matrix.

Template Parameters
_ScalarThe scalar type used for a density matrix element: real or complex.
Note
This type represents the (sqrt(2)-scaled) triplet (1,0) 1-DM.
