Detailed Description

template<typename _Shell>
class GQCP::PrimitiveCanonicalKineticEnergyIntegralEngine< _Shell >

A class that can calculate integrals over the canonical kinetic energy operator, i.e. - 1/2 nabla^2.

Template Parameters
_ShellThe type of shell that this integral engine is related to.
