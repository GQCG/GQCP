Detailed Description

A London-modified Cartesian GTO.

Note
Mathematically speaking, a Cartesian GTO is a complex-valued scalar function taking an Euclidean vector (3D-vector) as argument, which is why we inherit from ScalarFunction.
Calling operator() returns the value of the unnormalized London GTO.
Contracted London GTOs can be expressed as linear combinations of GTOs: LinearCombination<LondonCartesianGTO>.
