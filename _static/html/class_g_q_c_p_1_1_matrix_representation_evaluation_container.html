Detailed Description

template<typename _Matrix>
class GQCP::MatrixRepresentationEvaluationContainer< _Matrix >

A class that can efficiently construct the matrix representation of a second-quantized operator.

Template Parameters
_MatrixThe type that is used as a matrix representation, i.e. a container that stores the matrix representation.
